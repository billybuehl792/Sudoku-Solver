# Sudoku Solver
Solve sudoku puzzles with backtracking.

## Solve a Puzzle
Pass an array with 0s as unknowns.

```
import Sudoku

puzz_0 = [
    0,0,0,0,0,7,0,3,4,
    0,3,0,0,0,0,0,0,6,
    0,0,4,6,0,8,0,0,0,
    0,9,0,0,1,0,5,0,0,
    0,0,5,2,0,6,3,0,0,
    0,0,2,0,4,0,0,7,0,
    0,0,0,8,0,9,1,0,0,
    9,0,0,0,0,0,0,5,0,
    6,8,0,3,0,0,0,0,0,
]

puzz = Sudoku(puzz_0)
print(puzz.puzzle(solved=True))
```
